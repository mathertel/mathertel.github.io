Projects of mathertel on GitHub

This site contains public projects and documentation of the public projects and libraries made by Matthias Hertel.

Arduino Libraries

OneButton

An Arduino library for using a single button for multiple purpose input.

DMXSerial

An Arduino library for sending and receiving DMX packets.

Radio

An Arduino library to control FM radio chips like SI4703, SI4705, RDA5807M, TEA5767.

LiquidCrystal_PCF8574

A library for driving LiquidCrystal displays (LCD) by using the I2C bus and an PCF8574 I2C adapter.

DmxSerial2

An Arduino library for sending and receiving DMX RDM packets.

rfcodes

This is a RF signal encode and decode library that can be adapted to different protocols by specifying the timing conditions is a table. Can be used with 433 MHz and IR receivers and senders.

RotaryEncoder

RotaryEncoder Arduino Library
